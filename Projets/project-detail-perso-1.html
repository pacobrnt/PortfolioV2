Site E-commerce Moderne

Projet Personnel

Aperçu du Projet

Ce projet est une plateforme e-commerce complète développée avec les technologies modernes. Elle permet aux utilisateurs de parcourir des produits, de les ajouter au panier, et de finaliser leurs achats de manière sécurisée.

React Node.js MongoDB Stripe

Interface Principale

Description Détaillée

L'application offre une expérience utilisateur fluide avec une interface moderne et intuitive. Les principales fonctionnalités incluent :

  • Catalogue de produits avec filtres et recherche avancée
  • Système de panier dynamique avec calcul en temps réel
  • Intégration de paiement sécurisé via Stripe
  • Gestion des commandes et historique d'achats
  • Interface d'administration pour la gestion des produits
  • Système d'authentification et de profils utilisateurs

Processus de Paiement

Technologies Utilisées

Frontend: React, Redux, Tailwind CSS
Backend: Node.js, Express
Base de données: MongoDB
Paiement: Stripe API
